Programmation backend

Fonctionnalités principales :
Lecture du fichier d'entrée :
L'application reçoit en paramètre un fichier texte contenant une liste d'ISBN.
Elle extrait cette liste d'ISBN et la convertit en une chaîne de caractères.
Récupération des données produit :
La chaîne d'ISBN est utilisée pour effectuer une requête SQL vers une base de données.
La requête récupère une dizaine de paramètres spécifiques pour chaque produit correspondant aux ISBN fournis.
Génération des argumentaires :
L'application permet de générer des argumentaires produits de deux manières : En récupérant des argumentaires existants. En créant de nouveaux argumentaires à partir de modèles HTML (templates).
L'utilisateur peut choisir parmi plusieurs modèles HTML disponibles.
Fusion et création du PDF :
Tous les argumentaires produits, qu'ils soient récupérés ou créés, sont fusionnés en un seul document PDF.
L'application utilise les bibliothèques iTextSharp et iText 7 pour la manipulation et la création du fichier PDF.
Options utilisateur :
L'utilisateur a la possibilité de choisir les modèles HTML à utiliser pour la création des argumentaires.
Il peut également choisir de n'inclure que la première page de chaque argumentaire dans le fichier PDF final.
Technologies :
  • C#
  • iTextSharp
  • iText 7
  • SQL

En résumé :

L'application doit automatiser la création de documents PDF personnalisés en utilisant des données produits extraites d'une base de données et des modèles HTML. L'utilisateur a un contrôle flexible sur le contenu du PDF final.
